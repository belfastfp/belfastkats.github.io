Martin Thompson

@mjpt777

 CRDTs in the Wild

CRDTs are the cool new data structures in distributed computing. We hear of things like "Eventually Consistent" and "Monotonic Function" but what do they really mean? We also hear that functional programming is going to solve the concurrency problem but where is the evidence? What does it mean to put this theory into practice?

In this talk we will explore a CRDT that is core to the implementation of a messaging system which replicates a log of messages across multiple nodes. This data structure is functional in nature yet breaks with tradition and delivers performance that beats all imperative and functional data structures that have come before it.

About Martin

Martin is a consultant, trainer, and coach specializing in designing high-performance and low-latency systems.

Garth Gilmour

@GarthGilmour

 You Put The Lime In The Coconut - 10 best practises for Functional Programming with Java 8

Although Java 8 has been the official standard for some time, most developers are either not using it or using it in name only. Most enterprise organisations are just starting to explore the benefits of mixing the functional programming style with the well established Java platform. This talk offers 10 guidelines for making the best use of Java's new functional features whilst avoiding common traps and anti-patterns.

About Garth

Developer, trainer, martial artist, husband and father. Not in that order.

Tom Prior

@priortd

 Beginning Haskell: Dive In, Its Not That Scary!

This talk will be an overview of what I have learnt about Haskell over the past few months. It is aimed at dispelling the myth that Haskell is this hard, academic, scary thing so that the audience will go away encouraged to dive into Haskell themselves and see the beauty of the language. The talk won't go into advanced topics such as Monads, Functors etc., but it will cover basic Functional Programming concepts in terms of the Haskell language to show that the language is actually very accessible.

About Tom

Tom is one of the organizers of Functional Kats Cork. He's been a software developer for 5 and a half years and codes a lot in Scala in his current role at Newsweaver in Cork.
